## 구현할 기능 목록
  - 금액을 갯수로 변환
  - 갯수만큼 로또 생성
  - 로또 번호 범위 지정
  - 생성된 로또 번호들 출력
  - 당첨 번호 저장 
  - 보너스 번호 저장
  - 로또 안에 당첨 번호가 존재 하는지 확인
  - 일치하는 당첨 번호 갯수로 당첨 결과 확인
  - 당첨 결과 저장
  - 수익율 계산
  - 당첨 통계 출력
------------------------------------------
## 예외 상황
- 로또 번호 중복 방지
- 당첨 번호 중복 방지 (보너스 번호 포함)
- 당첨 번호 갯수 6개 확인
- 구입 금액 1000원 미만 금액이 남는지 확인
- 최대 10만원 까지만 구매 가능
